# Telegram Mini App — Каталог бьюти-мастера

## Структура проекта

```
tg-beauty/
├── CLAUDE.md              ← Этот файл: документация проекта
├── research.md            ← Исследование конкурентов
├── brief.md               ← Бриф: экраны, элементы, переходы
└── tg-app/                ← Mini App (фронтенд)
    ├── index.html         ← Точка входа, 4 экрана
    ├── styles.css         ← Стили: темы, карточки, анимации
    ├── app.js             ← Логика: навигация, рендер, Telegram API
    └── services.json      ← Данные услуг (менять здесь!)
```

## Экраны и навигация

```
1. Каталог        →  2. Детали услуги  →  3. Выбор даты/времени  →  4. Вы записаны!
   (профиль,          (карусель,           (календарь,               (карточка записи,
    портфолио,         цена, описание,      слоты времени,            вернуться)
    список услуг)      кнопка Записаться)   кнопка Записаться)
```

Все переходы анимированы (slide 250ms). Кнопка «← Назад» видна на каждом экране.

## Где менять данные

- **Услуги** — `tg-app/services.json` (id, name, description, price, duration, emoji)
- **Имя мастера** — `tg-app/index.html`, искать «Анна Иванова»
- **Адрес** — `tg-app/index.html`, искать «Тверская»
- **Портфолио** — `tg-app/app.js`, массив `portfolioItems`
- **Слоты времени** — `tg-app/app.js`, массив `timeSlots`
- **Выходные дни** — `tg-app/app.js`, свойство `isWeekend` в `getNextDays()`

## Как запустить

```bash
npx serve tg-app -l 3000
```

Открыть http://localhost:3000 — работает в браузере без Telegram.
